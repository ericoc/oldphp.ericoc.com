Indego Bike Share PHP Library

  • Started: November 16 (Monday), 2015
  • Langage: PHP
  • The Philadelphia Indego Bike Share program is interesting to me because of the technology behind it, including the fact that RFID tokens are used to check out bicycles.
  • When I realized that an API is available for the bike share stations, I just had to take advantage of it, so I created a PHP class/library to work with the data that the city makes available.

Zabbix Slack AlertScript

  • Started: December 27 (Friday), 2013
  • Langage: Bash
  • I had a chance to write a quick Bash script meant to forward alerts from the Zabbix monitoring application to the Slack messaging service as I couldn't find an already existing method by which to do so.
  • Since I couldn't find an already existing solution online, I figured I would offer it up to the public even if it is just a short/simple shell script for others to use in case they can't find any thing already online either.

gaw.sh URL Shortener

  • Started: December 2 (Friday), 2011
  • Language: PHP, MySQL, HTML, CSS, JavaScript
  • I originally purchased the domain name "gaw.sh" as a short address to remember quickly and to have some fun with.
  • However, I eventually realized that it would make a great domain name for a URL shortener so I began creating my own.
  • The source can be found on GitHub at https://github.com/ericoc/gaw.sh.
